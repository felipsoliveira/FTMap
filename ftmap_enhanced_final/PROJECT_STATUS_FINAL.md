## 🚀 Como Usar o Sistema Organizado

### Execução Simples:
```bash
cd ftmap_enhanced_final
./run_analysis.sh
```

### Execução com Proteína Customizada:
```bash
cd ftmap_enhanced_final
./run_analysis.sh minha_proteina.pdb
```

### Execução via Python:
```bash
cd ftmap_enhanced_final
python run_ftmap.py data/protein_prot.pdb --quick
```
